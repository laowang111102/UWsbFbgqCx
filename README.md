# 台球管理系统简介 java1089

## 项目概述

台球管理系统是基于SSM（Spring、SpringMVC、MyBatis）框架与Vue.js的前端技术构建的全新系统。该系统旨在提供方便、快捷的台球场地管理与会员服务。

## 系统角色

### 管理员角色

- 可以登陆后台，管理会员、台球桌、预约等信息。
- 能够查看详细数据，执行修改、删除、查询操作。
- 实时监控订单状态，并开启计费功能。

### 会员角色

- 可预约球桌，并查看预约详情。
- 进行商品购买与充值，管理个人账户余额。

## 运行环境要求

- JDK 8.0
- Tomcat 8
- MySQL 5.7
- Maven 3.5
- 开发工具：IntelliJ IDEA 或 Eclipse

## 技术栈

- 后端：Spring、SpringMVC、MyBatis
- 前端：Vue.js、Bootstrap、jQuery

## 目录结构

```
 project/
 ├── backend/                # 后端代码目录
 ├── frontend/               # 前端代码目录
 ├── db/                     # 数据库相关文件
 └── document/               # 项目文档
```

## 部署步骤

1. 环境搭建：配置JDK、Tomcat、MySQL及Maven环境。
2. 数据库部署：执行`db`目录下的SQL脚本进行数据库初始化。
3. 后端部署：编译后端代码，并部署至Tomcat服务器。
4. 前端部署：编译前端代码，放置到Tomcat服务器的相应目录。

## 核心亮点

- 使用SSM框架进行高效的后端开发。
- 系统模块化设计，方便后续功能扩展与维护。
- 实时的计费与订单状态监控，提升管理水平。
